## 🚀 Quick Start

### Prerequisites
*   **Node.js** 20+ & **pnpm**
*   **Ollama** (installed and running)
*   **Rust** (for Tauri builds)

### Installation
```bash
# Enter the sanctuary
cd Metamorphic-Resonance-Human-Language-Gamers

# Ignite the dependencies
pnpm install

# Pull the core model
ollama pull gemma3:4b

# Pull the core model
ollama pull embeddinggemma

# Launch the Language Engine
pnpm run tauri dev
```
